PAN

( Personal Area Network )

Jaringan komputer yang digunakan untuk komunikasi
antara perangkat pribadi atau menghubungkan ke
tingkat yang lebih tinggi dan jaringan internet.
( Info lengkap PAN terdapat di PDF )

LAN

( Local Area Network )

Jaringan komputer yang hanya mencakup wilayah kecil,
seperti jaringan komputer kampus, gedung, kantor,
dalam rumah, sekolah atau yang lebih kecil.
( Langkah kerja membuat jaringan LAN terdapat di PDF )

MAN

( Metropolitan Area Network )

Jaringan dalam suatu kota dengan transfer data
yang menghubungkan berbagai lokasi seperti kantor,
rumah, gedung, DLL

( Info lengkap MAN terdapat di PDF )

WAN

( Wide Area Network )

Jaringan komputer yang mencakup area besar seperti
jaringan komputer antar wilayah, kota, bahkan negara
dapat didefinisikan sebagai router dan saluran komunikasi publik.
( Info lengkap WAN terdapat di PDF )
